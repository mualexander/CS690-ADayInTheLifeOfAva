# CS690-ADayInTheLifeOfAva

## Business Requirement

A software package to help with planning trips by organizing itineraries, tracking bookings, and managing budgets.

## Use Cases
* UC1 Track locations
* UC2 Bookmark URLs (travel guides, blogs, resturant reviews, etc) by location
* UC3 Track flights for each location (prices, dates/times)
* UC4 Track hotel options in each location
* UC5 Budget reporting (running total of flight and hotel choices, with optional costs to be set for activities and food per location)
* UC6 Budget constraints: Allow for an overall budget to be set to help guide choices for flights, hotels, etc to stay within the budget
